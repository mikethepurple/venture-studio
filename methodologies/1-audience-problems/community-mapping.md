# Community Mapping

## When to use

Always runs in step 1 (wave 1). Finds 15-20 online communities where the target audience congregates. Maps the landscape — does not mine for problems.

## Inputs needed

- brief.md

## Process

1. **Search for communities by platform.** For the audience described in brief.md, find:
   - **Subreddits** — audience-specific (e.g., r/VintageClothing) and thematic (e.g., r/ThriftStoreHauls). Search Reddit for the audience's keywords.
   - **Facebook groups** — search Facebook group directory for audience terms.
   - **Discord servers** — search Disboard, Discord.me, and top.gg for relevant servers.
   - **Slack communities** — search Slofile and Google "[audience] slack community."
   - **Niche forums** — industry-specific boards, regional forums, Stack Exchange sites.
   - **Other platforms** — LinkedIn groups, Quora spaces, Twitter/X communities, Telegram channels.

2. **For each community found, record:**
   - Community name
   - Platform
   - URL (if public)
   - Size (members/subscribers)
   - Activity level (posts per day/week, or qualitative: high/medium/low)
   - Geo/language focus (if any)

3. **Target 15-20 communities.** Prioritize size and activity. Include at least 3 different platforms for diversity.

4. **Don't mine for problems.** Don't read posts. Don't extract complaints. Just map where the audience lives.
